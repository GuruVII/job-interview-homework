# Job-interview-homework
A widget created for a job interview

## Requirements

  * [Node.js](http://nodejs.org)

## Quickstart

  * Clone the repository

  * Install the Node modules required for development
```bash
npm install
```
  * Start Webpack
```bash
npm run webpack
```
